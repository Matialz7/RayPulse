# RayPulse

RayPulse یک داشبورد سبک برای مانیتورینگ اوتباندهای Xray است.

## چه چیزهایی را نشان می‌دهد
- delay و وضعیت اوتباندها
- بهترین اوتباند در لحظه
- پایدارترین اوتباندها
- کاربران آنلاین از روی access log
- ترافیک اوتباندها در صورتی که در `debug/vars` موجود باشد

## قابلیت‌ها
- اپلیکیشن تک‌فایل پایتون
- نصب به صورت سرویس systemd
- پشتیبانی از HTTPS با سرتیفیکیت خودت
- نصب‌کننده تعاملی
- حفظ تاریخچه نمودار حتی وقتی اوتباند down می‌شود

## نصب سریع از GitHub
بعد از اینکه پروژه را روی GitHub گذاشتی، کاربرها می‌توانند با یک دستور نصبش کنند:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/YOUR_USERNAME/RayPulse/main/install.sh)
```

## نصب از فایل ZIP
1. فایل ZIP را از GitHub دانلود کند.
2. روی سرور استخراج کند.
3. وارد پوشه پروژه شود.
4. این دستورها را بزند:

```bash
chmod +x install.sh
./install.sh
```

## فایل‌ها
- `raypulse.py` فایل اصلی برنامه
- `install.sh` نصب‌کننده تعاملی
- `uninstall.sh` حذف سرویس و فایل‌ها
- `RayPulse.service.example` نمونه فایل systemd

## مسیرهای پیش‌فرض
- Metrics URL: `http://127.0.0.1:11112/debug/vars`
- Access log: `/usr/local/x-ui/access.log`

## نکته‌ها
- اگر می‌خواهی روی پورت `443` اجرا شود، این پورت باید خالی باشد.
- اگر دامنه پشت CDN است، باید ارتباط origin با سرور برقرار بماند.
- اگر Xray آمار اوتباند را در `debug/vars` ندهد، ترافیک اوتباندها صفر می‌ماند.
